# input_materials

Put your local study materials here, then run:

```powershell
python scripts/generate_site.py
```

Supported in this first version:

- PDF
- Markdown / `.md`
- DOCX

Current limitations:

- No OCR for scanned PDFs.
- No handwritten note recognition.
- No image-only formula extraction.
- No PPTX or Jupyter Notebook parsing yet.
